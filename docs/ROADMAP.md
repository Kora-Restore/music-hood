# Roadmap

## Milestone 0: Spec + UX parity with Musicolet
- Define screens, flows, shortcuts
- Define library model and playlist format
- Define metadata strategy

## Milestone 1: Minimal Player
- Folder scan + search
- Play queue
- Basic playlists

## Milestone 2: Musicolet-grade UX
- Fast navigation, smart sorting, powerful search
- Tag editing, artwork, gapless, replaygain

## Milestone 3: Optional YouTube capture helper
- yt-dlp integration as a separate module
- Hotkey-driven "download from clipboard"

## Milestone 4: Packaging + releases
- Windows installer
- Linux AppImage/Flatpak
