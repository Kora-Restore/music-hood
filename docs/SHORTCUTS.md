# Keyboard Shortcuts Spec

## Global
- Space: Play/Pause
- Ctrl+Right / Ctrl+Left: Next/Previous track
- Ctrl+Up / Ctrl+Down: Volume up/down
- M: Mute
- Q: Toggle Queue panel
- Ctrl+K: Focus global search
- Esc: Clear search / close modal
- Ctrl+, : Settings

## Library navigation
- 1: Folders
- 2: Artists
- 3: Albums
- 4: Playlists
- / : Focus search (alt to Ctrl+K)

## Track list actions
- Enter: Play selection (play now)
- A: Add selection to queue
- Shift+A: Add selection to queue next (play next)
- Del: Remove from queue/playlist (when applicable)
- Ctrl+E: Edit tags (selection)
- Ctrl+S: Save selection as playlist

## Queue
- J/K: Move selection down/up (vim-ish optional mode)
- Ctrl+Up / Ctrl+Down: Move queued track
- X: Remove from queue
- Ctrl+L: Clear queue (with confirm)

## Import (optional)
- Ctrl+V: Paste link into import input
- Ctrl+Enter: Start import
- Ctrl+Shift+P: Import as playlist folder
